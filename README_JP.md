# nunu-name

ランダムで発音しやすい名前を生成し、必要に応じて数字サフィックスを付ける小さな CLI ツールです。

## 必要環境

- Rust nightly（このプロジェクトは `#![feature(random)]` を使用）

## ビルド

```bash
cargo build --release
```

## 使い方

```bash
cargo run -- [COUNT|MIN-MAX] [LETTERS|MIN-MAX] [DIGITS|MIN-MAX]
```

各位置引数は省略可能です。

- `COUNT`: 出力する名前の数（デフォルト: `1`）
- `LETTERS`: 各名前の文字数（デフォルト: `1`）
- `DIGITS`: 名前の末尾に付ける数字の桁数（デフォルト: `1`）

各引数は次のどちらかで指定できます。

- 固定の整数（例: `4`）
- 範囲（例: `3-8`）。実行ごとにランダムに選択

### ヘルプ

```bash
cargo run -- --help
# または
cargo run -- -h
```

### 実行例

```bash
# 1件生成、1文字 + 1桁（すべてデフォルト）
cargo run --

# 5件生成、各6文字 + 2桁
cargo run -- 5 6 2

# 3つの引数すべてを範囲でランダム化
cargo run -- 3-7 4-8 0-3
```

## 補足

- 不正な数値はデフォルト値にフォールバックします。
- 範囲指定は自動で正規化されるため、`8-3` は `3-8` と同じ挙動になります。
